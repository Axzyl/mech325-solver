üìê Reference equations from Section 2.1.3 Formulae. These are general formulas you can use for calculations.

Geometry Equations

$$p = \frac{\pi D_P}{N_P} = \frac{\pi D_G}{N_G} = \frac{\pi}{P_d}$$

Circular pitch

$$m = \frac{D_P}{N_P} = \frac{D_G}{N_G} = \frac{1}{P_d}$$

Module

$$m_G = \frac{N_G}{N_P}$$

Gear ratio

$$D_o = \frac{N + 2}{P_d}$$

Outside diameter

$$D_R = D - 2b$$

Root diameter

$$a = \frac{1}{P_d}$$

Addendum

$$b = \begin{cases} \frac{1.25}{P_d} & P_d < 20 \\ \frac{1.20}{P_d} + 0.002 & P_d \geq 20 \end{cases}$$

Dedendum (piecewise)

$$c = \begin{cases} \frac{0.25}{P_d} & P_d < 20 \\ \frac{0.20}{P_d} + 0.002 & P_d \geq 20 \end{cases}$$

Clearance (piecewise)

$$h_f = a + b$$

Whole depth

$$h_k = 2a$$

Working depth

$$t = \frac{p}{2} = \frac{\pi}{2P_d}$$

Tooth thickness

$$F_{nominal} = \frac{12}{P_d} \quad \left(\frac{8}{P_d} < F < \frac{16}{P_d}\right)$$

Face width (nominal)

$$C = \frac{D_P + D_G}{2} = \frac{N_P + N_G}{2P_d}$$

Center distance

$$D_b = \frac{N_p}{P_d} \cos \phi$$

Base circle diameter

Forces and Motion

$$F_{t,x-y} = W_t$$

Transmitted force

$$F_{r,x-y} = F_{t,x-y} \tan \phi$$

Radial force

$$T = W_t \frac{d}{2}$$
