Step 9: Geometry Factor J

📈 MANUAL LOOKUP - Look up J from bevel gear geometry factor graph

How to use the graph:
• Calculate J for both pinion and gear separately
• For pinion: Use pinion curve, x-axis = number of teeth in gear (NG)
• For gear: Use gear curve, x-axis = number of teeth in pinion (NP)
• Read J value from y-axis at the intersection
• Use the lower J value (more conservative) for stress calculations
(use lower of JP and JG)
